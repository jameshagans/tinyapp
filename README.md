## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Clone the project to a local repository on your device 
- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command or `npm start`.
